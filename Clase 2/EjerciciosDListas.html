5 etiquetas de HTML

Etiqueta <hr> :
Esta etiqueta dibuja una linea y luego produce un salto de linea, pudiendo separar un parrafo de texto, de otro, intercalando una etiqueta < hr > entre ambos.
Etiqueta < stong > :
Esta etiqueta define texto con fuerte importancia. Usualmente se vera en negrita.
Etiqueta: < del > :
El texto entre esta etiqueta, usualmente se mostrara tachado.
Etiqueta < sup > :
Indica que el texto se debe mostrar mas alto, y generalmetne mas pequeno que el tramo principal del texto, a modo de superindice.
Atributos de las Etiquetas <h1>, <h2>, <h3>, <h4>, <h5>, <h6>:

  • class
  • style
  • title
  • dir
  • lang
  • xml:lang
